## Суть простыми словами
Базовый Python: как объявлять переменные и типы, ветвить логику (`if`) и повторять действия (`for`). Для фронтендера почти всё знакомо — важны отличия: отступы вместо `{}`, `True/False` ведут себя как `1/0`, нет неявного приведения типов.

## Ключевое

### Типы и преобразования
- `int()`, `float()`, `str()`, `bool()` — приведение типов.
- `int(3.8)` → **3** (отбрасывает дробь, т.е. вниз). `round(3.8)` → 4 (к ближайшему; `round(x, 2)` — до 2 знаков).
- `bool(x)`: любое ненулевое/непустое → `True`, `0`/`''`/`None` → `False`.
- `True == 1`, `False == 0` → с ними можно считать: `sum([True, False, True]) == 2` (частый приём — посчитать «сколько прошло условие»).
- Нет неявного приведения: `'2' + 2` → `TypeError`. Склеивать можно только строку со строкой.

### Строки в выводе
- f-строки: `f'{x} руб.'`. Тройные кавычки `"""..."""` — многострочный текст.
- `"=" * 20` — повтор строки. `print(a, b, sep=' - ', end='!')` — разделитель/окончание вместо `\n`.

### Условия
- `if условие:` + отступ 4 пробела. Операторы: `and`, `or`, `not`.
- Сравнение разных типов (`'20' >= 20`) → `TypeError` (str vs int).
- Строковые предикаты → `bool`: `isdigit()` (только цифры), `isalpha()` (только буквы), `islower()`, `isspace()`.

### Циклы
- `for i in range(5)` → 0..4. `range(3, 7)` → 3,4,5,6 (**правая граница не включается**).
- `break` — выйти из цикла; `continue` — пропустить итерацию; `pass` — заглушка (ничего не делает, цикл идёт дальше).
- `zip(a, b)` — идёт параллельно по спискам: `for x, y in zip(a, b)`. Останавливается на самом коротком.
- `enumerate(a)` → пары `(индекс, значение)`: `for i, v in enumerate(a, start=1)`. Лучше, чем `range(len(a))`.
- `sum(список)` — сумма; `math.prod(список)` — произведение (не встроенная, из `math`).

## Пример
```python
# посчитать покупателей через bool, собрать id из цифр строки
is_buyer = [True, False, True, True]
print(sum(is_buyer))            # 3

order = ''
for ch in '00A1642':
    if ch.isdigit():
        order += ch             # '001642'
```

## Грабли
- `range(a, b)` **не включает** `b`.
- Не менять переменную-итератор внутри `for` (`price = price + 50` не меняет список) — заводи новую переменную.
- `int('123.5')` → ошибка (точка только у float); сначала `float()`, потом при желании `int()`.
- Забыть, что сравнение `str` и `int` падает с `TypeError`.
