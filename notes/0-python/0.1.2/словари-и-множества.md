# Словари и множества (dict, set)

> Приложение к [0.1.2](README.md). Ещё две встроенные коллекции: словарь (пары ключ→значение) и множество (уникальные значения).

## Словарь `dict` — {ключ: значение}
Быстрый доступ по ключу (не по индексу). Ключи уникальны.

### Создание
```python
d = {'США': 'Вашингтон', 'Италия': 'Рим'}
d = dict(name='Стефан', age=49)              # через dict(...)
d = dict(zip(keys_list, values_list))        # из двух списков
```

### Доступ и поиск
- `d[key]` — значение; если ключа нет → `KeyError`.
- `d.get(key)` → `None`, если нет (без ошибки); `d.get(key, default)` — своё значение по умолчанию.
- `key in d` — проверка наличия ключа.

### Изменение
- `d[key] = value` — добавить новый или обновить существующий.
- `d.pop(key)` — удалить и вернуть значение.
- `d.update(other_dict)` — влить другой словарь.

### Итерация
- `d.keys()` → `dict_keys`, `d.values()` → `dict_values`, `d.items()` → пары `(ключ, значение)`.
- Обернуть в `list()`, чтобы получить обычный список. В цикле: `for k, v in d.items():`.

## Множество `set` — {уникальные, неупорядоченные}
Нет дубликатов, нет порядка, нет доступа по индексу. Быстрая проверка «есть ли элемент».

### Создание
```python
s = {1, 2, 3}          # {1, 2, 2, 3} даст то же самое — дубликаты отбрасываются
empty = set()          # ПУСТОЕ — только так! {} это пустой dict, не set
set('ананас')          # {'а','н','с'} — уникальные символы
set(['сыр','сыр','сок'])   # {'сыр','сок'} — убрать дубли из списка
```

### Изменение
- `add(x)` — добавить. `remove(x)` — удалить (ошибка, если нет). `discard(x)` — удалить без ошибки.
- `pop()` — удалить и вернуть **произвольный** элемент (порядок не определён).

### Операции над множествами (символ или метод)
| Операция | Символ | Метод |
|---|---|---|
| Объединение | `\|` | `union()` |
| Пересечение | `&` | `intersection()` |
| Разность | `-` | `difference()` |
| Симметр. разность | `^` | `symmetric_difference()` |

- Сравнение `<` / `>` — строгое под-/надмножество (пустое `set()` меньше любого непустого). Сравнивать можно **только set с set** (иначе ошибка); `==`/`!=` — разрешены между разными типами.
- `len(s)` — число уникальных (частый приём: `len(set(values))`). `sorted(s)` → отсортированный **список** (само множество не сортируется, `.sort()` нет).

## Грабли
- `{}` — это **словарь**, пустое множество только `set()`.
- `{5, 5.0} == {5}` → `True`; `5` и `5.0` считаются одним элементом (как и в ключах словаря).
- `set` неупорядочен — не полагайся на порядок вывода и на то, что вернёт `pop()`.
- `d[key]` бросает `KeyError` — если ключ может отсутствовать, бери `d.get(key, default)`.
