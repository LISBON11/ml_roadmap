## Суть простыми словами
Три последовательности. **Список** `[]` — изменяемый (добавляй/удаляй/сортируй). **Кортеж** `()` — неизменяемый, легче в памяти, для «зафиксированных» данных. **Строка** — тоже последовательность символов, работает как список для чтения (индексы, срезы), но неизменяема. Плюс две коллекции без порядка-по-индексу: **словарь** (ключ→значение) и **множество** (уникальные значения).

## Ключевое

### Списки `[]` (изменяемые)
- Срезы `[start:end:step]` (как у строк). `list * n` — повтор: `[1,2]*3 → [1,2,1,2,1,2]`.
- Удаление: `pop(i)` — по **индексу**, возвращает удалённый элемент; `remove(val)` — по **значению** (когда индекс неизвестен).
- Добавление: `append(x)`, `insert(i, x)`, `extend(other)` или `list1 + list2`.
- `sort()` (по возрастанию), `sort(reverse=True)` (по убыванию), `min()`/`max()`.

### Кортежи `()` (неизменяемые)
- Создаются круглыми скобками: `t = (42.35, -71.06)`. Элементы разных типов — можно.
- `val in t` → `bool`; `t.count(val)` — сколько раз встречается.
- **Распаковка**: `x, y = present_box` — присвоить несколько переменных за раз (удобно для данных из API/БД).

### Строки (неизменяемые)
- Индексация/срезы: `s[:7]`, `s[::2]` (каждый 2-й), `s[::-1]` (**реверс**).
- `list(s)` → список символов; `sep.join(list)` → склеить обратно (`'.'.join([...])`).
- Методы: `replace(a, b)`, `find(sub)` / `rfind(sub)` (последнее вхождение; **`-1` если не найдено**), `upper()`, `lower()`, `strip()` (убрать пробелы по краям).
- Спецсимволы: `\n` перенос, `\t` табуляция, `\\` слеш.

## Пример
```python
s = '00A1642 Кизильник, 50 кустов'
print(s[:7])            # '00A1642' — срез
print(s[::-1][:5])      # реверс

nums = [3, 2, 5, 4, 1]
nums.sort(reverse=True) # [5,4,3,2,1]
first = nums.pop(0)     # 5, список стал [4,3,2,1]
```

## Грабли
- `pop(i)` — индекс, `remove(val)` — значение. Перепутать = не тот элемент/ошибка.
- Строку и кортеж **нельзя** менять на месте — только пересобрать в новую.
- `find()` возвращает `-1` (не ошибку), когда подстроки нет — проверяй.
