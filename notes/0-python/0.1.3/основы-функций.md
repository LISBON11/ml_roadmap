# Основы функций: def, return, аргументы

> Приложение к [0.1.3](README.md). Базовый синтаксис функций. Про область видимости — в самом [README](README.md), задачи — в [задачи.md](задачи.md).

## Объявление и вызов
```python
def show_metric(name, value):     # def имя(параметры):
    print(f"{name}: {value}")     # тело — с отступом
show_metric("CR", "3.6%")         # вызов
```

## `return` — вернуть результат
- `return value` — отдаёт значение наружу.
- **Нет `return`** → функция возвращает `None` (даже если внутри были вычисления). Присвоишь результат — получишь `None`.

## Аргументы
- **Позиционные** — связываются по порядку: `f("a", "b")`.
- **Именованные** — по имени, порядок неважен: `f(value="b", name="a")`.
- Смешивать можно, но **сначала позиционные, потом именованные**.

### Значения по умолчанию
```python
def calc(amount, discount=0.05):      # дефолт — амортизирует необязательные параметры
    ...
calc(1000)            # discount = 0.05
calc(1000, 0.2)       # discount = 0.2
```
- Параметры с дефолтом идут **только в конце**, после всех обязательных. Иначе `SyntaxError`:
```python
def calc(discount=0.05, amount):   # ❌ non-default argument follows default argument
```
- ⚠️ Изменяемый дефолт (`def f(x, box=[])`) создаётся один раз и копит состояние между вызовами — см. [задачи.md](задачи.md), №6/17. Фикс: `box=None` + `if box is None: box = []`.

## Встроенные функции и методы
- Часто нужные: `len`, `sum`, `min`, `max`, `sorted`, `zip`, `list`, `type`, `print`, `help(func)` (справка без интернета).
- Ленивые объекты (`zip`, `range`, `enumerate`) не хранят всё сразу; `list(zip(a, b))` — материализовать, чтобы взять `len()` или посмотреть.
- Метод вызывается **на объекте**: `s.upper()`, `lst.append(x)`. Функция — сама по себе: `len(lst)`.
- Не у всех объектов есть нужное свойство → преобразуй тип (`list(zip_obj)`), иначе ошибка.

### Мини-шпаргалка методов
- Списки: `append`/`extend`/`insert`/`remove`/`pop`/`clear`, `sort`/`reverse`, `index`/`count`.
- Строки: `replace`/`upper`/`lower`/`strip`, `find` (−1 если нет), `split`/`join`, предикаты `isdigit`/`isalpha`/`islower`/`isspace`.

## Математика
- `**` — степень: `2 ** 3 = 8`. Корень степени `n` = возведение в `1/n`: `x ** 0.5` — квадратный корень.
- Точнее/специальнее — модуль `math` (`math.sqrt`, `math.prod`, …); подключать `import math`.

## Грабли
- Забыть `return` → функция «работает», но результат `None`.
- Дефолтный параметр перед обязательным → `SyntaxError`.
- `zip`/`range` — ленивые: `print(zip(a,b))` покажет `<zip object ...>`, а не данные; оборачивай в `list()`.
