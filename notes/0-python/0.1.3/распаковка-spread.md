# Распаковка `*` / `**` — аналог spread/rest из JS

> Мостик из JS: `[...arr]`, `{...obj}`, `fn(...args)`, `const [x, ...rest]` — всё это в Python делается звёздочками `*` (итерируемое) и `**` (словари). Тесно связано с [[0.1.2]] (структуры данных) — там же shallow-копии.

## Суть простыми словами
`*` «разворачивает» любой **итерируемый** объект (list, tuple, str, range, set…) поэлементно; `**` разворачивает **словарь** в пары ключ-значение. Та же звёздочка слева от `=` или в сигнатуре функции работает наоборот — **собирает** остаток в список (rest-паттерн / `*args`).

## Ключевое

### Spread — разворачивание (как `[...arr]`)
```python
arr = [1, 2, 3]
copy   = [*arr]            # [1, 2, 3]  — shallow-копия
merged = [*a, *b]          # конкатенация двух списков
more   = [*arr, 4, 5]      # добавить элементы
chars  = [*"abc"]          # ['a','b','c'] — в JS так не выйдет
nums   = [*range(3)]       # [0, 1, 2]

# словари — двойная звёздочка (аналог {...obj})
merged = {**d1, **d2}      # объединить; правый перезаписывает ключи левого
```

### Spread в аргументы функции (как `fn(...args)`)
```python
def add(x, y, z): ...
add(*arr)                  # = add(1, 2, 3)
f(**{"x": 1, "y": 2})      # именованные аргументы из словаря
```

### `*args` / `**kwargs` в сигнатуре (как `function f(...args)`)
```python
def f(*args, **kwargs):
    # args   — tuple позиционных
    # kwargs — dict именованных
    ...
```

### Rest-паттерн (как `const [first, ...rest] = arr`)
```python
first, *rest = [1, 2, 3, 4]      # first=1, rest=[2,3,4]
*init, last  = [1, 2, 3, 4]      # init=[1,2,3], last=4
head, *mid, tail = [1, 2, 3, 4]  # 1, [2,3], 4  — звёздочка максимум одна
```

## Таблица соответствий JS → Python
| JS | Python |
|----|--------|
| `[...arr]` | `[*arr]` (или `arr.copy()` / `list(arr)`) |
| `{...obj}` | `{**obj}` (или `obj.copy()` / `dict(obj)`) |
| `[...a, ...b]` | `[*a, *b]` (для списков ещё `a + b`) |
| `fn(...args)` | `fn(*args)` |
| `const [x, ...rest] = a` | `x, *rest = a` |
| `function f(...args)` | `def f(*args)` |

## Грабли (где отличается от JS)
- `[*arr]` — **shallow**-копия (как и `[...arr]`). Для глубокой: `import copy; copy.deepcopy(arr)`.
- `*` работает с **любым итерируемым**, `**` — **только со словарём** (не путать области применения).
- В rest-паттерне звёздочка может быть **только одна**: `a, *b, *c = ...` → SyntaxError.
- `*rest` всегда собирает в **list**, даже если распаковывали tuple.
- Порядок в `{**d1, **d2}`: при конфликте ключей побеждает **последний** (как spread объектов в JS).
