## Суть простыми словами
Производная показывает, **как быстро меняется функция в конкретной точке**. Если знаем, как функция меняется в одной точке, — можем заменить маленький участок её графика прямой линией.

Производная — ответ на вопрос **«сколько единиц $y$ приходится на одну единицу изменения $x$?»**, или: **«какой наклон у графика в этой точке?»** — где наклон считается не от начала координат $(0,0)$, а по **касательной**, проведённой в точке $x$.

Она **не говорит, чему равен $y$**. Она говорит, **во сколько раз изменение $y$ больше изменения $x$**.

**Аналогия с дорогой.** Производная — это **уклон дороги прямо под твоими ногами**:
- посмотришь на участок длиной 1 км → узнаешь **средний уклон** всей дороги;
- посмотришь на участок длиной 1 см → почти узнаешь уклон **именно в этой точке**.

Поэтому:
- $\dfrac{\Delta y}{\Delta x}$ с большой $\Delta x$ → средняя скорость изменения **на участке**;
- $\dfrac{\Delta y}{\Delta x}$ при $\Delta x \to 0$ → производная **в точке**.

Именно поэтому в формуле появляется предел: мы уменьшаем шаг всё меньше и меньше, чтобы «поймать» настоящий наклон в одной точке.

## Ключевое / формулы

**Строгая формула**, из которой всё получается:

$$f'(x) = \lim_{\Delta x \to 0} \frac{f(x + \Delta x) - f(x)}{\Delta x}$$

Читается: *производная — это предел отношения изменения $y$ к изменению $x$, когда изменение $x$ стремится к нулю.* Разберём по кусочкам:

| Кусочек | Что это |
|---|---|
| $f(x + \Delta x) - f(x)$ | насколько изменилась функция, то есть $\Delta y$ |
| $\Delta x$ | насколько мы сдвинулись по оси $x$ |
| $\dfrac{\Delta y}{\Delta x}$ | скорость изменения |
| $\lim$ (предел) | делаем шаг всё меньше и меньше, пока не подойдём к точке практически вплотную |

- **Линеаризация** — приближение сложной нелинейной функции линейной, чтобы решать её средствами линейной алгебры. Локально график $\approx$ прямая (касательная). Формула:

$$f(x) \approx f(a) + f'(a)\,(x - a)$$

  Буквально: **новое значение $\approx$ старое известное значение $+$ (скорость изменения $\times$ насколько сдвинулись)**. Это то же самое, что $\Delta y \approx f'(a)\cdot \Delta x$, только $\Delta x$ записали как $(x-a)$, а $\Delta y$ сразу прибавили к известному значению: $\text{новое } y \approx \text{старое } y + \Delta y$.

## Пример / код

### 1. Производная $f(x) = x^2$ руками (через предел)
Наклон касательной = предел наклона секущей, когда шаг $h \to 0$:

$$
\begin{aligned}
f'(x) &= \lim_{h \to 0} \frac{(x+h)^2 - x^2}{h}
       = \lim_{h \to 0} \frac{x^2 + 2xh + h^2 - x^2}{h} \\[4pt]
      &= \lim_{h \to 0} \frac{2xh + h^2}{h}
       = \lim_{h \to 0} (2x + h) = 2x
\end{aligned}
$$

В точке $x = 3$: производная $f'(x) = 2x$, подставляем → $f'(3) = 6$.

**Что значит эта шестёрка?** Если немного увеличить $x$ около $3$, то $y$ будет расти примерно **в 6 раз быстрее**.

### 2. Численная проверка (что производная не врёт)
| Шаг | Значение |
|---|---|
| Было | $x = 3,\quad y = 9$ |
| Чуть увеличили | $x = 3.01$ |
| Тогда | $y = 3.01^2 = 9.0601$ |
| Изменение $y$ | $9.0601 - 9 = 0.0601$ |
| Изменение $x$ | $3.01 - 3 = 0.01$ |
| Отношение | $\dfrac{0.0601}{0.01} = 6.01$ |

Почти $6$ — ровно то, что сказала производная. ✔

### 3. Линеаризация на числах ($f(x) = x^4$ в точке $a = 3$)
Здесь $f(a) = 3^4 = 81$, а $f'(a) = 4\cdot 3^3 = 108$. Сдвигаемся на $\Delta x = -0.001$:

$$f(2.999) \approx f(3) + f'(3)\cdot(-0.001) = 81 + 108\cdot(-0.001) = 80.892$$

То есть формула линеаризации — это буквально запись идеи **«новое значение = старое значение + изменение»**. (Проверка: $2.999^4 = 80.8920\ldots$ — совпадает.)

### 4. Связь с ML (градиентный спуск)
Обучение = минимизация функции потерь $L(w)$. Производная $L'(w)$ говорит наклон в текущей точке — куда и как круто «вниз». Шаг делаем **против** наклона — $w \leftarrow w - \eta \, L'(w)$, где $\eta$ — learning rate:

```python
# один шаг градиентного спуска для L(w) = w**2 (минимум в w=0)
def grad(w):        # L'(w) = 2w  — та самая производная x²
    return 2 * w

w = 5.0             # старт
lr = 0.1            # learning rate — длина шага
for _ in range(20):
    w = w - lr * grad(w)   # идём против наклона
print(round(w, 4))         # ≈ 0.0576 — сползли к минимуму
```

Производная $2w$ здесь — это буквально «наклон чаши потерь» в точке $w$. Чем круче наклон, тем больше шаг; у дна ($w \to 0$) наклон $\to 0$, и шаги затухают.

## Грабли (где ошибался)
- **Производная $\ne$ значение функции.** $f'(x)$ — это наклон, а не $y$. В точке $x=3$ для $x^2$: значение $f(3)=9$, а наклон $f'(3)=6$ — разные числа.
- **Наклон меряется по касательной, а не «из нуля».** Не $y/x$, а $\Delta y / \Delta x$ на бесконечно малом участке вокруг точки.
- **Линеаризация — приближённая.** Отношение дало $6.01$, а не ровно $6$: чем больше $\Delta x$, тем сильнее промах. Точное значение получается только в пределе $\Delta x \to 0$.
- **Знак важен.** В градиентном спуске шаг делается $w - \eta \, L'(w)$, то есть **против** наклона; поставишь $+$ — уедешь от минимума.
- **Слишком большой $\eta$** — перепрыгиваешь минимум и расходишься; слишком маленький — ползёшь вечно.
