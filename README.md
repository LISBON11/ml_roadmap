# 📝 Заметки по дорожной карте

Здесь живут твои заметки — **папка на каждую тему** из трекера. У каждой темы в трекере (`index.html`) есть иконка 📝, которая ведёт ровно в её папку, на файл-«шапку» `README.md`.

## Как это работает
- **Одна тема = одна папка:** `notes/{папка_этапа}/{код_темы}/`. Внутри — файл-шапка `README.md`, на который ведёт 📝. Например: тема `S.4` → [`notes/sql/S.4/README.md`](notes/sql/S.4/README.md); тема `1.1.3` → `notes/1-math/1.1.3/README.md`.
- **К теме можно много заметок.** В папке темы держи сколько угодно файлов: `README.md` — главная (суть + оглавление), рядом отдельные разборы (`pitfalls.md`, `2026-07-20-практика.md`, …), картинки — в подпапке `images/`. Ссылки на них складывай в раздел **Заметки** внутри `README.md`.
- **Заводишь по ходу**, не сразу все. Клик по 📝 ведёт на путь; если файла ещё нет, GitHub предложит создать его прямо там.
- **Формат** — по [`notes/_шаблон.md`](notes/_шаблон.md). Пример заполненной шапки: [`notes/sql/S.4/README.md`](notes/sql/S.4/README.md).

## Папки этапов (для ссылок 📝)
| Код в трекере | Папка |
|---|---|
| `S.x`, `A.x` (SQL) | `notes/sql/` |
| `0.x` (Python) | `notes/0-python/` |
| `1.x` (Математика) | `notes/1-math/` |
| `2.x` (Классический ML) | `notes/2-ml/` |
| `3.x` (Глубокое обучение) | `notes/3-dl/` |
| `4.x` (AI-инженерия) | `notes/4-ai/` |
| `5.x` (Деплой/найм) | `notes/5-deploy/` |

Итоговый путь: `notes/{папка_этапа}/{код_темы}/README.md`. Пример: тема `1.2.2` → `notes/1-math/1.2.2/README.md`.

## Настройка ссылок в трекере
В трекере кнопка **«📝 База»** задаёт, куда ведут ссылки:
- `notes/` (по умолчанию) — относительные пути. Удобно, когда открываешь заметки локально в VS Code.
- Ссылка на GitHub, напр. `https://github.com/USER/REPO/blob/main/notes/` — тогда 📝 открывает красиво отрендеренную шапку темы прямо на GitHub. **Рекомендую этот вариант** после того, как зальёшь репозиторий.
