# 🧭 Дорожная карта: Frontend → AI Engineer

Персональный план перехода из фронтенд-разработки в роль **AI Engineer** (LLM-приложения: RAG, агенты, продакшн). Построен в два слоя: **[A]** — критический путь до найма, **[B]** — глубокое понимание «изнутри».

> Реалистичный срок при ~20–25 ч/нед: до готовности подаваться — **~7–9 мес**; полная программа со слоем [B] — **~12–15 мес**.

## 🗺️ С чего начать

1. Прочитай **[обзор и карту](00_обзор_и_карта.md)** — стратегия, слои A/B, порядок прохождения.
