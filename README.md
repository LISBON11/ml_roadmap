# 📝 Заметки по дорожной карте

Здесь живут твои заметки — по одному md-файлу на тему. Из **трекера** (`index.html`) у каждой темы есть иконка 📝, которая ведёт ровно на соответствующий файл здесь.

## Как это работает
- Путь к заметке: `notes/{папка_этапа}/{код_темы}.md`. Например, тема `S.4` → [`notes/sql/S.4.md`](sql/S.4.md); тема `1.1.3` → `notes/1-math/1.1.3.md`.
- Файлы создаёшь **по мере прохождения** — не нужно заводить все сразу. Клик по 📝 в трекере ведёт на путь; если файла ещё нет, GitHub предложит создать его прямо там.
- Формат — по [`_шаблон.md`](_шаблон.md). Пример заполненной: [`sql/S.4.md`](sql/S.4.md).

## Настройка ссылок в трекере
В трекере кнопка **«📝 База»** задаёт, куда ведут ссылки:
- `notes/` (по умолчанию) — относительные пути. Удобно, когда открываешь заметки локально в VS Code.
- Ссылка на GitHub, напр. `https://github.com/USER/REPO/blob/main/notes/` — тогда 📝 открывает красиво отрендеренную заметку прямо на GitHub. **Рекомендую этот вариант** после того, как зальёшь репозиторий.
